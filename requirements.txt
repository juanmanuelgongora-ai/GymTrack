================================================================================
COMANDOS DE INSTALACIÓN RÁPIDA
================================================================================

Instalación Completa:
  1. Instalar dependencias de PHP en backend:
     cd backend && composer install
  
  2. Instalar dependencias de Node.js en backend:
     cd .. && npm install
  
  3. Instalar dependencias de Node.js en frontend:
     cd frontend && npm install

O ejecutar el script de setup (si existe):
  npm run setup (desde raíz del proyecto backend)
